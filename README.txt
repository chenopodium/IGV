Installation for development:
=============================
- Download NetBeans 7.2+ (www.netbeans.org/downloads JavaSE version)
- Source classes are in src folder
- include all jars in the lib folder
- the main class is org.broad.igv.ui.Main

Running Standalone Version:
===========================
- the igv.jar is in the webfolder
- run it with: java -Xmx2000m -jar igv.jar

(substitute 2000M to the amount of memory you can/want to give IGV - note in case you are using Jvav 32 bit, you will be limited to about 1.2G)
